Apilite - Express TypeScript Template

Template backend RESTful API menggunakan ultimate-express dan TypeScript dengan fitur autentikasi dan manajemen database

Sistem Autentikasi

Sistem autentikasi lengkap dengan fitur register dan login menggunakan JWT

Manajemen Database

Database SQLite3 dengan Knex.js untuk query builder dan migration tool

TypeScript

JavaScript dengan penambahan tipe data untuk pengembangan yang lebih aman

Getting Started

1. Clone Repository

git clone https://github.com/arghoritma/apilite.git

2. Install Dependencies

npm install

3. Configure Environment

# Create .env file with: PORT=3000 JWT_SECRET=your-secret-key

4. Start Development Server

npm run dev

API Endpoints

Register new user account

POST /api/users/register

{ "name": "John Doe", "email": "john@example.com", "password": "password123" }

Login to existing account

POST /api/users/login

{ "email": "john@example.com", "password": "password123" }

Mendapatkan profil pengguna (memerlukan autentikasi)

GET /api/users/profile
